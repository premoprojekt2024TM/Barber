## Deployment

Hogy elínditsa ezt a projektet

```bash
    cd ./server
```
Az envexample alapján töltse ki az .env fájlt
```bash
    cp .envexample .env
```
Build
```bash
    docker compose build
```
Up
```bash
   docker compose up  
```

# Az alábbi portokon érhetőek el az imagek:
- **localhost:8080** - Backend
- **localhost:3000** - Frontend
- **localhost:5432** - Adatbázis
